## Overview

This repository hosts a Tableau dashboard designed to analyze the usage of the Bellabeat Leaf smart device. The dashboard focuses on various wellness trends, including feature adoption, time-based activity, sleep patterns, user wellness profiles, and correlation insights. 

## Features

- **Feature Adoption**: Understand which features users engage with the most.
- **Time-Based Activity Analysis**: Track user activity over time to identify trends.
- **Sleep Patterns**: Analyze sleep data to improve user wellness.
- **User Wellness Profiles**: Create profiles based on user data for personalized insights.
- **Correlation Insights**: Discover relationships between different wellness metrics.

To start using the Bellabeat Product Dashboard, you can download the latest release from the [Releases section](https://github.com/Arlos59/bellabeat-product-dashboard/releases). Follow the instructions below to set up the dashboard on your local machine.
